# 🚀 快速部署到 Netlify

## 第一步：修改 API 地址

1. 打开 `index.html` 文件
2. 找到第 1042 行：`const API_BASE = "https://your-backend-api.com";`
3. 将其改为你的实际后端 API 地址，例如：
   ```javascript
   const API_BASE = "https://api.example.com";
   ```

## 第二步：上传到 Netlify

### 方法 1：拖拽上传（最简单）

1. 将整个 `netlify-deploy` 文件夹压缩成 ZIP 文件
2. 访问 https://app.netlify.com 并登录
3. 点击 **"Add new site"** → **"Deploy manually"**
4. 将 ZIP 文件拖拽到上传区域
5. 等待部署完成（1-2 分钟）

### 方法 2：通过 Git 仓库

1. 将这个文件夹推送到 GitHub/GitLab
2. 在 Netlify 点击 **"Import from Git"**
3. 选择你的仓库
4. 设置：
   - **Build command**: 留空
   - **Publish directory**: `netlify-deploy`
5. 点击 **"Deploy site"**

## 第三步：完成！

部署完成后，Netlify 会给你一个类似这样的地址：
```
https://your-site-name.netlify.app
```

访问这个地址就能看到你的网站了！

---

## ⚠️ 重要提醒

1. **必须修改 API 地址**：部署前一定要把 `index.html` 中的 `API_BASE` 改成你的后端地址
2. **后端需要支持 CORS**：确保后端允许跨域请求
3. **建议使用 HTTPS**：如果后端是 HTTP，可能会遇到混合内容问题
