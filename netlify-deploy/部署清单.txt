部署前检查清单
========================================

□ 1. 已修改 index.html 中的 API_BASE 地址
   （找到 const API_BASE = "https://your-backend-api.com"; 这一行）
   
□ 2. 确认后端服务正常运行且可以访问

□ 3. 确认后端已配置 CORS 允许跨域请求

□ 4. 准备好 Netlify 账号（GitHub/GitLab/Bitbucket 登录）

========================================
快速部署步骤
========================================

1. 修改 API 地址
   打开 index.html，修改第 1039 行的 API_BASE

2. 打包文件夹
   将 netlify-deploy 文件夹压缩成 ZIP

3. 上传到 Netlify
   - 访问 https://app.netlify.com
   - 点击 "Add new site" → "Deploy manually"
   - 拖拽 ZIP 文件上传

4. 等待部署完成
   通常 1-2 分钟，Netlify 会自动分配域名

5. 测试功能
   访问分配的域名，测试所有功能是否正常

========================================
重要提示

⚠️ 部署前必须修改 API 地址！
⚠️ 确保后端支持 CORS 跨域！
⚠️ 建议后端也使用 HTTPS！
